CSS селекторы
a.b. - одновременно есть классы a и b
пробел - вложенный селектор (потомки)
+ - некст сиблинг
а>б - б строго внутри родителя а
а[атрибут] - тег с атрибутом
* – любые элементы.
без пробелов - и то и другое должно быть
div ~ p – правые соседи: все p на том же уровне вложенности, которые идут после div.
-------
приоритеты селекторов:
идентификаторы стоят 100;
классы стоят 10;
селекторы тега стоят 1.

Переборы
for (let z in lala) - дает ключи (все ключи, даже с пустыми значениями)
for (let z of lala) - дает значения (в старом ES был только он и только для массивов) дает только те элем-ы, в которых есть значения
array.forEach(функция) - перебор массива дает значения
array.map( х => х * 2) - создает новый массив где каждый элемент обработан функцией, в качестве аргументов 
может быть 1 значение, 2 ключь, 3 изначальный массив (в примере х - это значение)
